This project is my first complex project from the Website Building module at Link Academy. It's a personal CV website built with HTML, CSS, JavaScript, and Webpack. I used Bootstrap for responsive design and added visual effects with some libraries. I used Figma only for editing images and the logo.

Current features:

  • Smooth navigation
  • Scroll animations
  • Back to top button
  • Interactive image gallery
  • Responsive layout for all devices

Future features:

  • Translation to multiple languages
  • Dark mode
