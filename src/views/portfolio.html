The Phonebook Application is a JavaScript-based web application that allows users to store, display, and manage contact information. The application enables users to save names along with their corresponding phone numbers and view them in an organized list.

The user interface is divided into two main sections:

  • A form for adding new contacts, which includes input fields for the contact's name and phone number, along with a submit button.
  • A list that displays all saved contacts in a clear and organized manner.

The Phonebook Application is implemented using React, with Vite used as the development and build tool.
